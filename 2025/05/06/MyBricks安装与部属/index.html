MyBricks安装与部署

‌**MyBricks是一个低代码开发平台，主要用于企业级中后台系统的搭建。**‌

MyBricks支持多端适配，包括电脑、平板和手机，适用于各种场景如CRM、数据大屏、企业官网等。它通过低代码的形式生产组件，这些组件不仅适用于低代码项目，还可以在源码项目中使用。MyBricks支持微信、支付宝、抖音等小程序搭建，并能轻松连接大模型、企业微信等第三方平台，提供全面的解决方案‌

✅ 一. 环境需要

  • 已安装有 Node 14+（推荐v14.21.0）、npm或者yarn等包管理工具，推荐 npm 7.x 以上
  • 配置并启动了数据库环境，MySql 5.7 以上，如果安装的是 MySQL8.x，注意密码加密方式设置为：Legacy Password Encryption，切记不要设置为 Strong Password Encryption
  • 云服务器【我用的阿里云】
  • 远程软件【我用的FinalShell】
